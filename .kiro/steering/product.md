# Product Context

## What This Workspace Is

This is Richard Williams' personal productivity and marketing operations workspace — not a traditional code repository. It powers "The Body," an 11-organ system for context management, decision-making, execution tracking, calibration, and willpower development.

## The Body System

The Body is a structured context architecture with 11 organs, each owning a specific domain:

- **brain.md** — Strategic priorities, decision log, leverage framework
- **eyes.md** — Market data, performance metrics, dashboards
- **hands.md** — Execution tracking, active work
- **memory.md** — Relationship graph, institutional knowledge
- **spine.md** — Session bootstrap, tool access, key IDs
- **heart.md** — Autoresearch loop protocol, experiment queue
- **device.md** — Tools, automation, recurring task templates
- **nervous-system.md** — Signal routing, notification management
- **amcc.md** — Streak tracking, hard thing, avoidance detection
- **gut.md** — Word budgets, compression rules, system health
- **body.md** — Navigation layer mapping the whole system

Navigation starts at `~/shared/context/body/body.md`.
