Autoresearch Engine

How the body system maintains itself. Inspired by Karpathy's autoresearch: 630 lines, 700 experiments, measurable results. Small, fast, autonomous, compounding.

The body is an 11-organ system that holds context, makes decisions, tracks patterns, and enforces willpower. Each organ is a self-contained markdown file with a word budget enforced by the gut. The autoresearch loop runs experiments on these organs autonomously — snapshot the current state, apply a modification (compress, add, restructure, remove, reword, merge, or split), evaluate with two independent blind subagents, and keep or revert based on accuracy and completeness scores.

The primary metric is usefulness per token: does the organ answer the questions it is supposed to answer, accurately and self-containedly, without requiring additional file reads? Word count is a constraint (ceiling), not an objective. An organ at 95% of budget that answers everything correctly is fine. An organ at 50% that misses questions needs content added, not celebrated for being small.

After 0 loop runs, the system has run 2 experiments with a 100% keep rate. 4,821 words have been saved through compression while maintaining accuracy thresholds (100% for Brain and Memory, 95% for Eyes and Hands, 90% for all others). The body currently holds 16,094 words against a 21,000-word ceiling, leaving 4,906 words of headroom for future content.
