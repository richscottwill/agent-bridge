// After CanonChart.render() completes, add aria attributes + synthesize a summary
function annotateChartA11y(canvasId, summary) {
  const c = document.getElementById(canvasId);
  if (!c) return;
  c.setAttribute('role', 'img');
  c.setAttribute('aria-label', summary);
}

18 Scrub-the-chart interaction (Robinhood Advanced Charts) R2
Source: Robinhood Advanced Charts launched a scrubbing feature that lets users compare securities at specific points in time — hover a chart point and the KPI area updates inline. Translation: when hovering any week on the trend chart, the entire page updates to show that week's KPIs + three-question cards + variance + narrative. Release hover → everything returns to the selected week. Turns the chart into a time-scrubbing lens on the whole page.
