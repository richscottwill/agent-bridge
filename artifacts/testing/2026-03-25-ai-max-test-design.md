# AI Max Test Design — US Market

Due: 3/28/2026

---

## Hypothesis

AI Max's expanded reach and automated asset generation will increase registrations by 15-25% without degrading CPA beyond 10% of current baseline, when applied to NB campaigns in the US market.

## Test Parameters

| Parameter | Value |
|-----------|-------|
| Market | US |
| Campaign type | NB (Non-Brand) |
| Test duration | 4 weeks minimum |
| Traffic split | 50/50 (AI Max vs manual control) |
| Primary metric | Registrations |
| Secondary metrics | CPA, CTR, impression share, search term quality |
| Guardrails | CPA must not exceed 110% of control. If CPA exceeds 120% for 7 consecutive days, pause and review. |

## Measurement Framework

1. Baseline period: 2 weeks pre-test (current NB performance)
2. Test period: 4 weeks with 50/50 split
3. Incrementality: compare test vs control on matched keyword sets
4. Quality check: review search terms generated by AI Max weekly — flag irrelevant queries

## Guardrails

- CPA ceiling: 110% of control average
- Rollback trigger: 120% CPA for 7 consecutive days
- Budget: match control spend exactly (no budget advantage)
- Cannibalization check: monitor Brand campaign performance during test for leakage
- Weekly review cadence with Google (Mike Babich)

## Coordination

- Google: Mike Babich (external) — AI Max feature access, best practices
- Internal: Stacey (US campaign owner), Brandon (approval)
- Other Amazon teams: check if Retail/Flex have tested AI Max — benchmark

## Success Criteria

| Outcome | Action |
|---------|--------|
| Regs +15% AND CPA within 110% | Scale to 100% NB, plan EU rollout |
| Regs +10-15% AND CPA within 110% | Extend test 2 more weeks for confidence |
| Regs flat or CPA >110% | Revert. Document learnings. |

The 15% reg lift threshold is calibrated to OCI's US result (+24%). We're setting a lower bar because AI Max is less proven. If it hits even 10%, it's worth extending.

## Open Questions
- [ ] Does AI Max respect our existing negative keyword lists?
- [ ] How does AI Max interact with OCI bidding?
- [ ] Can we exclude specific asset types (e.g., auto-generated headlines)?
- [ ] What's the minimum campaign budget for AI Max to learn effectively?

These questions should be answered in the Google sync with Mike Babich before test launch. If any answer is a dealbreaker (e.g., AI Max ignores negative keywords), the test design needs revision.
