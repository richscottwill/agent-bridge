# AI resources
[][1][Richard Williams][1][][2][Peter Ocampo][2]
#### **Amazon Internal**[GenAI Tools - List][3]
#### Cedric:[][4]General-purpose AI Assistant
https://console.harmony.a2z.com/internal-ai-assistant
* Cedric browser extension to analyze Quip document
* Load files into Cedric and ask questions about the content
* Create Personas (pre-prompt and files)
* Data Collections (Cedric references files as part of the prompt.
 * Can ask it to summarize across multiple Quips (within the Collection). This is a manual process for now because of access.
#### Q: Chat-based AI Assistant
https://ask.qbusiness.aws.dev/#/chat
* Integrated with Amazon and AWS tools, and has access to internal wikis, etc
* Useful for investigating topics related to Amazon or coming up with ideas using Q Apps
 * Q Apps “Agents”: Idea starters
  * Experiment ideas
  * Text ad ideas
*Cedric is more versatile and better at handling complex tasks, while Amazon Q is optimized for quick, fact-based business queries

#### Scenario 1: Learn about a topic
* Load files into Cedric, and you can ask questions about the files
* Have Cedric access the web to learn about something, like how to use Index Match in Excel (sometimes faster than a Google search)
* Use Cedric extension to ask questions about a Quip. Example of a denser topic: [AB SSR Registration MIE Central Wiki][5]
#### Scenario 2: Critique writing
* Cedric website: Load Amazon Writing Conventions doc into a persona, and have the persona critique the doc.
* Cedric Extension: Analyze the current page you’re on, and ask for critiques (can upload Amazon Writing Conventions doc)
#### Scenario 3: Synthesize content
* Load related files into Cedric, and ask it to create writing that draws from the different sources
#### Drawbacks:
* AI isn’t great at math/data analysis yet
* AI is only as good as the source it’s drawing from (garbage in, garbage out), so you still have to use your judgment
 * This is quickly becoming less of a drawback

**Ideas for future:**
* Office Hour Persona: Create and maintain Cedric Persona that could supplement regular Flash and Wiki (other teams can have their questions answered)
 * Slack channel bot, if they develop that more (Cedric or Bedrockbot)
