# Archive

Files moved here are no longer actively maintained but kept for reference.

## Categories

- context/ — archived context files, session logs, meeting notes
- research/ — completed or abandoned research, experiment docs
- tools/ — deprecated scripts and tool configs
- misc/ — documents, emails, CSVs, PDFs that don't fit above
- 2026-03/ — month-based overflow (legacy, will be sorted)
- hedy-syncs/ — archived Hedy sync outputs

## When to Archive vs Delete

- Archive: has historical value, might be referenced, contains decisions or results
- Delete: temp files, 0-byte files, debug output, duplicate exports
