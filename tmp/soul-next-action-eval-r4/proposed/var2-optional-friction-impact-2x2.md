## Optional Friction-Impact 2x2 Tool

When the agent faces **multiple viable actions** and is unsure which to prioritize, it may (but is not required to) mentally apply the following 2x2:

- **High Impact + Low Friction** → Default choice (do immediately)
- **High Impact + High Friction** → Break into smaller steps or schedule
- **Low Impact + Low Friction** → Only do if it protects a habit loop
- **Low Impact + High Friction** → Reject or delegate

This tool is **optional** and should only be used when it adds clarity. It is not required on every task.
