## Leverage Cascade (Test Version 5)

When determining the next best action, run this cascade in order:

1. **Current daily Leverage Move** (from latest brief) — Is it still valid? If yes → do it.
2. **Highest Five Levels opportunity** visible in current.md or rw-tracker.md.
3. **Biggest friction point** that is blocking multiple future actions.
4. **Protection of an existing habit loop** (if one is at risk of breaking).
5. **Only then**: Any other useful task.

**Final filter**: After identifying the top item, ask: "Is there any single smaller move that would make this even higher leverage?" If yes, choose the smaller move instead.
