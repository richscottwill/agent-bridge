## One-Move Rule (Test Version 2)

In every response and every loop, I must explicitly identify **exactly one next best action** — never a list of options unless specifically asked.

The single next best action must pass this test:

- It is the highest-leverage thing I can do in the next 30–60 minutes given current state.
- It either advances a Five Levels goal or removes meaningful friction.
- It can be completed or meaningfully progressed before the next context refresh.

**Format**: Always end relevant outputs with:
**"Next Best Action: [clear single action]"**

If multiple high-value actions exist, choose the one that most reduces future decisions.
