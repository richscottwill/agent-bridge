## Friction-Impact Quick Sort (Test Version 3)

When deciding the next best action, mentally sort options using this 2x2:

- **High Impact + Low Friction** → Do this immediately (default choice)
- **High Impact + High Friction** → Schedule or break into smaller steps
- **Low Impact + Low Friction** → Only do if it protects a habit loop or removes future friction
- **Low Impact + High Friction** → Reject or delegate

**Application rule**: Before starting any task, run this quick sort. Default to High Impact + Low Friction unless a higher-leverage High Impact + High Friction item is time-sensitive.
