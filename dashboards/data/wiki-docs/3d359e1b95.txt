# OCI Planning - Phased Rollout

Master planning document for OCI (Offline Conversion Import) phased rollout across US, UK, and DE. Contains tracking template configurations, rollout timelines, campaign splits for A/B testing, measurement methodology, and performance data.

Key sections:
- Tracking Template changes (URL construction, suffix, application phases)
- US OCI Planning (4-phase NB rollout, Brand rollout, timeline)
- UK OCI Planning (campaign splits, timeline, ROAS targets)
- DE OCI Planning (campaign splits, timeline)
- Measurement framework (baseline, control vs test, success metrics)
- Performance data reference tables
